# Bookmark Manager Frontend

A minimal, clean bookmark manager application for saving and organizing bookmarks from LinkedIn, Twitter/X, and articles.

## Getting Started

1. Install dependencies: `npm install`
2. Configure .env: `cp .env.example .env`
3. Run dev server: `npm run dev`

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + Radix UI
- Firebase Auth
- React Router + TanStack Query

## Project Structure

- `src/components/` - Reusable UI components
- `src/pages/` - Page components (Landing, Login, Dashboard)
- `src/services/` - API client and services
- `src/contexts/` - React contexts (Auth)
- `src/lib/` - Utilities and Firebase config
